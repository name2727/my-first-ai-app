\# 我的第一个 AI 应用



一个简单的 Python 脚本，通过 DeepSeek API 实现命令行交互式问答。



\## 功能



\- 连续提问，输入“退出”结束程序

\- 调用 DeepSeek 大模型回答问题



\## 运行方法



1\. 安装 Python 3.8 或更高版本

2\. 安装依赖库：

pip install requests



3\. 获取 DeepSeek API Key（https://platform.deepseek.com/）

4\. 将 API Key 填入 `rag\_demo.py` 文件中的 `api\_key = "你的key"`

5\. 运行脚本：

python rag\_demo.py
